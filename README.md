```bash

```

2. Open `index.html` in your web browser:
```bash
cd zakkath-calculator-malayalam
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```

No build process required - it's a static HTML page!

## üéØ Usage

1. Enter the quantity (if applicable) and value for each category
2. The calculator automatically computes individual Zakat amounts
3. Click "Get Sub Total" to calculate individual category totals
4. Click "Get Full Total" to get your total Zakat obligation
5. Use "Print" button to print your calculation
6. Use "Clear All" to reset all values

### Keyboard Navigation

- Use **Tab** key to move between input fields quickly
- All input fields support keyboard navigation for efficiency
